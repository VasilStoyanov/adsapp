# Project AdsHub

## Setup

1) Install NodeJS
2) Install node version manager (nvm)

##### Change directory to project root folder
```bash
$ cd adsapp
```
##### Use right version of NodeJS
```bash
$ nvm use
```

##### Install dependencies
```bash
$ npm i
```

##### Start the application
```bash
$ npm start
```

##### Running unit tests
```bash
$ npm test
```
